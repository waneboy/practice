Чтобы перейти к:
1. Первой странице (Каталог) - надо нажать на кнопку "Каталог" вверху страницы
2. Второй странице (Главная) - надо нажать на Логотип в шапке сайта
3. Третьей странице - надо перейти на седьмую страницу и нажать на любую карточку с надписью «Самое ценное в нашем мире — 
овощи» (P.s. остальные карточки оставляют пользователя на прежней странице) 
4. Четвертой странице - надо нажать на кнопку "Смотреть все" рядом с разделом "Популярные категории"
5. Пятой странице - надо нажать на кнопку "Овощи" в вверху страницы где находятся разные разделы с продуктами
6. Шестой странице - надо перейти на пятую страницу и нажать на любую карточку с морковкой 
7. Седьмой странице - надо нажать на кнопку "Все новости" на главной странице рядом с разделом "Новости"
